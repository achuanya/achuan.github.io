## 博客目录结构

    achuanya.github.io   Jekyll博客根目录
    ├─_drafts            文章草稿目录
    │
    ├─_includes          博客主要模块目录
    │  ├─nav.html        菜单栏目、文章列表、文章展示模块
    │  ├─header.html     资源加载模块
    │  ├─share.html      文章分享模块
    │  ├─disqus.html     Disqus第三方评论模块
    │  ├─copyright.html  博客底部模块，含Github组件、友联
    │  ├─css.html        CSS资源链接
    │  ├─scripts.html    JavaScript资源链接
    │
    ├─_layouts           文章布局目录
    │  ├─default.html    文章, 章节列表
    │  ├─post.html       文章声明
    │
    ├─assets             静态资源目录
    │  ├─article         文章图片
    │  ├─css             css样式
    │  ├─font            网页字体
    │  ├─image           网站图片
    │  ├─js              Javascript库
    │
    ├─pages              其他功能页面目录
    │  ├─rss.xml         Rss订阅
    │  ├─test.html       站点地图
    │  ├─ie.html         < IE8不支持
    │  ├─404.html        404页面
    │
    ├─.gitignore         Git忽略提交规则
    ├─_config.yml        配置文件
    ├─CNAME              重定向文件
    ├─index.md           个人介绍首页
    ├─LICENSE            MIT协议
    ├─README.md          自述文件
    ├─robots.txt         robots协议
    ├─sitemap.xml        sitemap协议
